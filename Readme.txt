Bixler Nails Studio - Gestión de Turnos

Descripción

Este proyecto es una aplicación web para la gestión de turnos en **Bixler Nails Studio**, un negocio de manicura. Permite a los clientes reservar citas para servicios específicos y almacena los datos de cada cliente en `localStorage`. La aplicación también proporciona una forma de visualizar las reservas realizadas de forma organizada en la consola de JavaScript.

Funcionalidades

- **Reserva de Turnos**: Los clientes pueden seleccionar un servicio, fecha y hora para reservar una cita.
- **Almacenamiento en `localStorage`**: Toda la información de las citas se guarda localmente en el navegador, lo que permite persistir los datos entre sesiones.
- **Visualización en la Consola**: Los detalles de los clientes y sus reservas se muestran en la consola de JavaScript, organizados por cliente y presentados en formato de tabla.

Tecnologías Utilizadas

- **HTML5**: Estructura básica de la aplicación.
- **CSS3**: Estilización de la interfaz de usuario.
- **JavaScript**: Lógica de la aplicación, manipulación del DOM, y gestión de `localStorage`.
